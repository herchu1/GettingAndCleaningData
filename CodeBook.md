### Code Book

* `subject`

    The subjects of the studies.
    
    An integer number from 1 to 30 that identifies each subject.

* `activity`

    The name of the activities being measured.
    
    A string that can be one of the following values:
    * WALKING
    * WALKING_UPSTAIRS
    * WALKING_DOWNSTAIRS
    * SITTING
    * STANDING
    * LAYING
